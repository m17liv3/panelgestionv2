M17LIV3 - Nombres actuales en renovaciones y balance

Cambios:
- Ingresos / renovaciones muestra el nombre actual del cliente si se ha editado.
- Balance muestra el nombre actual del cliente en ingresos por renovación.
- Exportaciones de renovaciones y balance usan el nombre actual.
- Al editar el nombre de un cliente, se sincroniza cliente_nombre en la tabla renovaciones para registros anteriores.
- No requiere SQL nuevo.
