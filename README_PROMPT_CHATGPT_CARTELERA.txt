M17LIV3 - Prompt automático para ChatGPT + subida final

Cambios:
- En Cartelera se sustituye el generador interno por un flujo con ChatGPT.
- Tú solo pegas los eventos del día, sin fecha.
- La app genera automáticamente la fecha actual en formato español.
- La app crea un prompt profesional para pegar en ChatGPT.
- Botón para copiar el prompt.
- Botón para subir la imagen final descargada de ChatGPT directamente a HORARIOS MUNDIAL.
- No requiere SQL nuevo.

Nota:
La subida final usa Supabase Storage en el archivo fijo imagen_actual.jpg.
