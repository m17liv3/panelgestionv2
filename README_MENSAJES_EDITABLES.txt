M17LIV3 - Mensajes rápidos editables

Novedad:
- En Msg aparece el botón "Editar plantillas de mensajes".
- Puedes editar Datos acceso, Aviso caducidad, Renovación OK y Cliente caducado.
- Las variables como {nombre}, {usuario}, {password}, {expiracion} se rellenan automáticamente con cada cliente.
- Si ejecutas SQL_MESSAGE_TEMPLATES.txt, las plantillas se sincronizan en Supabase.
- Si no ejecutas el SQL, la app guarda una copia local en el navegador.
