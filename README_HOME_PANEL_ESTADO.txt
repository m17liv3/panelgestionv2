M17LIV3 - Pantalla principal como panel de estado

Cambios:
- La pantalla principal muestra solo:
  - Activos
  - Caducan pronto
  - Expirados
  - Pagos pendientes
- Activos es solo informativo y no abre nada.
- Caducan pronto, Expirados y Pagos pendientes abren una ventana propia.
- Desde esas ventanas se puede:
  - Ver ficha
  - Renovar
  - Msg
  - Editar
  - Abrir más opciones
- Se elimina la sección Prioridad / Requieren atención.
- No se muestran clientes directamente en la pantalla principal.
- El botón inferior Clientes sigue abriendo la vista deslizante.
- No requiere SQL nuevo.
