Imagen fija APP

Nuevo apartado añadido al menú, justo encima de Cartelera del dia:
- Imagen fija APP

Uso:
1. Ejecuta una sola vez el SQL: SQL_STORAGE_IMAGEN_FIJA_APP.txt
2. Sube esta versión a GitHub.
3. Entra en la web-app y pulsa el botón ↻ actualizar.
4. Menú > Imagen fija APP.
5. Sube la imagen nueva del día.
6. Copia el link fijo una sola vez y pégalo en tu app Android.

El link fijo será el public URL de Supabase Storage para:
Bucket: cartelera
Archivo: imagen_actual.jpg

Cada vez que subas una imagen nueva, se sustituye el archivo anterior y el link no cambia.
