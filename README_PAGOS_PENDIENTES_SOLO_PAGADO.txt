M17LIV3 - Pagos pendientes simplificado

Cambio:
- En la ventana de Pagos pendientes, cada cliente muestra solo el botón "Pagado".
- Al pulsar "Pagado", pregunta importe cobrado y método de pago.
- Después de marcarlo como pagado, se actualiza la lista y desaparece de pagos pendientes.
- No requiere SQL nuevo.
